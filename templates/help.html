How to use this GUI

  1. You must update the SW Login credentials on the "SW Login" page, as they will be used to access all switches. Ensure that the same user/password exists on each switch..
  2. This webpage is using API call to get/set config on the switches - https (443) port should be allowd between the server and the switches.
  3. Add new switch under the "Add SW" page, Another option is to edit the "switches.yaml" with the relevant hostname/IP.
  4. You can set new config for any port on the "SW List" page by changing the config, checking the update button, and clicking update
